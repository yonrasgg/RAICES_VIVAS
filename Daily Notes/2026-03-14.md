## 📊 Métricas del Día

```sqlseal
TEMPLATE
| 📊 Métrica | Valor |
|---|---|
{{#each data}}
| ✅ Completadas hoy | {{this.done_today}} |
| 🔄 En progreso | {{this.in_progress}} |
| 🚫 Bloqueadas | {{this.blocked}} |
| 📈 Progreso global | {{this.pct}}% ({{this.total_done}}/{{this.total_tasks}}) |
{{/each}}

SELECT
  SUM(CASE WHEN status = 'done' AND completed = @date THEN 1 ELSE 0 END) as done_today,
  SUM(CASE WHEN status IN ('in-progress', 'review') THEN 1 ELSE 0 END) as in_progress,
  SUM(CASE WHEN status = 'blocked' THEN 1 ELSE 0 END) as blocked,
  COUNT(*) as total_tasks,
  SUM(CASE WHEN status = 'done' THEN 1 ELSE 0 END) as total_done,
  ROUND(100.0 * SUM(CASE WHEN status = 'done' THEN 1 ELSE 0 END) / MAX(1, COUNT(*))) as pct
FROM files
WHERE (type = 'task' OR type = 'subtask') AND path LIKE '05-Sprints%'
```

## 📝 Notas e Ideas

### Investigación de conectividad territorial

Geovanny presentó al equipo los hallazgos de la investigación de conectividad en territorios indígenas costarricenses:

**Fuentes consultadas:**
- SUTEL — Mapas de cobertura móvil 2024
- FONATEL — Informes de avance CPC 2023-2024
- MEP — Diagnóstico de centros educativos en territorios

**Hallazgos principales:**
- De los 24 territorios indígenas de Costa Rica, **17 tienen cobertura 3G/4G parcial** (señal intermitente, depende de ubicación)
- **7 territorios tienen conectividad nula o casi nula** (Chirripó, Alto Chirripó, Tayní, Telire, China Kichá zona alta, Salitre zona alta, Cabagra zona alta)
- La velocidad promedio donde hay 3G es de **0.5-2 Mbps** — suficiente para sync de JSON pero no para streaming de video
- Dispositivos más comunes: smartphones Android gama baja (1-2 GB RAM), algunos tablets del MEP

**Implicaciones arquitectónicas:**
- Confirma la necesidad de **offline-first** como requisito no negociable (RNF-01)
- El sync engine debe ser **oportunista**: sincronizar cuando detecte conexión, sin depender de ella
- Los contenidos multimedia deben ser **pre-descargados y comprimidos** (imágenes < 200KB, audio < 1MB)
- Se recomienda **CouchDB con replicación incremental** para minimizar tráfico
