# ✅ T-045: Indicador visual de estado de sincronización

## Objetivo

Implementar el componente `SyncIndicator` en el header de la aplicación para que el usuario vea en todo momento si está conectado, sincronizando o trabajando offline. Esto es clave para usuarios en zonas con conectividad intermitente.

## Checklist

- [x] Componente con 4 estados visuales: sincronizado (verde), sincronizando (azul pulsante), offline (gris), error (rojo)
- [x] Integración con hook `useSync()` para estado reactivo
- [x] Tooltip con detalle: última sincronización exitosa, pendientes en cola
- [x] Accesible: contraste AA, texto alternativo, no depende solo de color
