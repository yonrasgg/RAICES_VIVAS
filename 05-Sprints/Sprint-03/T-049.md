# ✅ T-049: Vista Gestión de Docentes

## Objetivo

Desarrollar la pantalla de gestión de docentes del módulo EDU. Permite registrar, editar y desactivar docentes comunitarios vinculados a programas educativos en territorios indígenas, cumpliendo con [[RF-EDU-01]].

## Criterio de la Rúbrica

> **RF-EDU-01 — Registro de docentes comunitarios**
> El sistema debe permitir registrar y gestionar docentes con datos de contacto, territorio asignado, idiomas que domina y programas vinculados.

## Checklist

- [x] Listado de docentes con búsqueda por nombre y filtro por territorio
- [x] Formulario de registro: nombre, contacto, territorio, idiomas, programas
- [x] Edición de datos de docente existente
- [x] Desactivación lógica (no eliminación física) con motivo
- [x] Almacenamiento en PouchDB con schema en `types/edu.ts`
- [x] Validación de campos requeridos
