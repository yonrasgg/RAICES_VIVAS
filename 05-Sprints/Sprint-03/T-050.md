# ✅ T-050: Vista Dashboard EDU con métricas

## Objetivo

Crear el dashboard del módulo educativo que presente indicadores clave del estado de los datos educativos. Cumple con [[RF-EDU-06]] y sirve como pantalla principal del módulo.

## Criterio de la Rúbrica

> **RF-EDU-06 — Indicadores del módulo educativo**
> El sistema debe mostrar métricas resumidas del módulo educativo: conteo de materiales, docentes, distribución por idioma y tipo.

## Checklist

- [x] Cards con métricas: total materiales, total docentes, materiales por idioma
- [x] Gráfico de distribución por tipo de material (usando CSS, no librería externa)
- [x] Datos consultados en tiempo real desde PouchDB con hook `useDB()`
- [x] Responsive: grid adaptable a móvil y desktop
- [x] Estado vacío con mensaje guía cuando no hay datos
