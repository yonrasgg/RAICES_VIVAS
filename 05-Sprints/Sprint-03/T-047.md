# ✅ T-047: Selector de idioma con persistencia

## Objetivo

Crear el componente de selección de idioma que permita al usuario elegir entre español, bribri, cabécar y ngäbere. La selección debe persistir en `localStorage` y aplicarse reactivamente a toda la interfaz mediante i18next.

## Checklist

- [x] Componente dropdown/select con banderas o iconos representativos
- [x] Integración con `i18next.changeLanguage()` reactivo
- [x] Persistencia en `localStorage` con detección de idioma previo al cargar
- [x] Accesible: etiqueta ARIA, navegable con teclado
- [x] Ubicación: integrado en el Header de AppShell
