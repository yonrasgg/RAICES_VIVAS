## Descripción

Crear el diagrama de contexto (C4 nivel 1) del sistema Raíces Vivas, identificando actores externos, sistemas adyacentes y flujos principales.

## Criterios de Aceptación (DoD)

- [ ] Diagrama C4 nivel 1 en Mermaid (archivo: `04-Arquitectura/Diagramas/C4-Contexto.md`)
- [ ] 5 actores externos identificados y documentados
- [ ] 3 módulos del sistema representados (EDU, SAB, SAL)
- [ ] Sistemas adyacentes definidos (MEP, CCSS, Servicio de Sincronización)
- [ ] Flujos principales documentados con flechas y descripciones

## Plan de Ejecución

> **Nota**: Esta tarea se PLANIFICA, aún no se ejecuta el diagrama final.

### Actores Externos Identificados

| # | Actor | Descripción | Interacción Principal |
|---|-------|-------------|----------------------|
| 1 | **Docente Comunitario** | Educador en escuelas de territorios indígenas | Carga materiales, revisa avances, practica ejercicios |
| 2 | **Auxiliar de Salud** | Personal EBAIS o brigadas de salud | Registra pacientes, consulta historial, programa citas |
| 3 | **Líder Cultural / Portador de Saberes** | Awá, sĩ'ö, anciano/a comunitario/a | Registra saberes, gestiona consentimiento, clasifica nivel de acceso |
| 4 | **Administrador de Sistema** | Equipo técnico (Geovanny/Equipo) | Configura módulos, gestiona usuarios, monitorea sincronización |
| 5 | **Miembro de Comunidad** | Residente del territorio indígena | Consulta materiales públicos, accede a información cultural aprobada |

### Sistemas Adyacentes

| Sistema | Tipo | Integración |
|---------|------|-------------|
| **MEP (Ministerio de Educación)** | Externo | Lineamientos curriculares, Decreto 40932 educación intercultural |
| **CCSS (Caja Costarricense)** | Externo | Estándares de registro de salud, datos EBAIS |
| **Servicio de Sincronización** | Interno | CouchDB sync entre servidor local (Raspberry Pi) y nube |
| **Sistema de Autenticación** | Interno | JWT + roles comunitarios (4 niveles per ADR-009) |

### Estructura del Diagrama C4

```
[Docente Comunitario] → [Raíces Vivas: Módulo EDU] → [MEP]
[Auxiliar de Salud]   → [Raíces Vivas: Módulo SAL] → [CCSS]
[Líder Cultural]      → [Raíces Vivas: Módulo SAB]
[Admin]               → [Raíces Vivas: Transversal]
[Todos]               → [Sync Service] ↔ [CouchDB Cloud]
```

### Herramienta y Destino

- **Formato:** Mermaid `C4Context` o `flowchart` en Markdown
- **Archivo destino:** `04-Arquitectura/Diagramas/C4-Contexto.md`
- **Bloqueado por:** Nada (esta es la tarea raíz)
- **Bloquea:** T-022, T-023, T-024 (modelos ER dependen del contexto)
