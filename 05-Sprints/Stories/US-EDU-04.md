# US-EDU-04: Motor de práctica educativa interactiva

## Descripción

Como docente comunitario, necesito un módulo de ejercicios interactivos en lengua indígena, para que los estudiantes practiquen lectura, escritura y vocabulario de forma guiada.

## Criterios de Aceptación

- [ ] Mínimo 3 tipos de ejercicio: selección múltiple, completar palabra, emparejar imagen-palabra
- [ ] Ejercicios disponibles offline
- [ ] Retroalimentación inmediata (correcto/incorrecto)
- [ ] Registro de progreso por estudiante
