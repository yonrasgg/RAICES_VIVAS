# US-SAL-05: Alertas de seguimiento clínico

## Descripción

Como auxiliar de salud, necesito recibir alertas automáticas cuando un paciente requiere seguimiento (medicación, control, vacunación), para no perder oportunidades de atención en comunidades con visitas poco frecuentes.

## Criterios de Aceptación

- [ ] Alertas configurables por tipo: medicación, control, vacunación, embarazo
- [ ] Notificación visual al abrir el módulo SAL
- [ ] Lista de alertas pendientes con prioridad (urgente/normal)
- [ ] Funciona offline, alertas basadas en fecha local
