## Descripción

Como **admin comunitario**, necesito clasificar los saberes ancestrales por categorías editables (agricultura, medicina tradicional, artesanía, etc.) para facilitar su organización y consulta.

## Criterios de Aceptación

- [ ] El admin puede crear, editar y eliminar categorías
- [ ] Las categorías soportan jerarquía (padre/hijo)
- [ ] Al registrar un saber, se puede asignar a una o más categorías
- [ ] La lista de categorías es visible en el módulo de búsqueda
- [ ] Las categorías se sincronizan entre dispositivos
