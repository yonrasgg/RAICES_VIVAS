## Descripción

Como **admin comunitario**, necesito que todo registro de saber ancestral requiera un consentimiento informado previo del portador, para garantizar el cumplimiento de los principios CARE y la legislación vigente.

## Criterios de Aceptación

- [ ] No se puede registrar un saber sin consentimiento aprobado
- [ ] El consentimiento puede ser oral (grabación) o escrito (firma digital/foto)
- [ ] Se registra: tipo, fecha, vigencia, si es revocable
- [ ] El portador puede revocar el consentimiento en cualquier momento
- [ ] La revocación desactiva el acceso al saber asociado
- [ ] Se genera log de auditoría para cada acción sobre consentimientos
