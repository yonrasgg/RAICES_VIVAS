## Descripción

Como **auxiliar de salud (ATAP)**, necesito exportar el expediente clínico de un paciente en formato compatible con EDUS para eliminar la transcripción manual que toma 3+ horas.

## Criterios de Aceptación

- [ ] El auxiliar puede seleccionar un paciente y exportar su expediente
- [ ] Se puede elegir rango de fechas y tipo de datos a exportar
- [ ] El sistema mapea campos locales a campos EDUS según tabla de correspondencia
- [ ] Se muestra previsualización con campos mapeados y advertencias de incompatibilidad
- [ ] Se soportan formatos CSV estándar CCSS y HL7 FHIR
- [ ] Exportación acumulativa disponible para brigadas (múltiples pacientes)
- [ ] Los datos se exportan cifrados y se registra la exportación en log de auditoría
- [ ] Funciona offline generando archivo para transferencia manual
