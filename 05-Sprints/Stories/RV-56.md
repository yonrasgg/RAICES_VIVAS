## Descripción

Como **docente comunitario**, necesito consultar un resumen de aciertos por tema y período de cada estudiante, para identificar áreas donde necesitan refuerzo.

## Criterios de Aceptación

- [ ] El docente puede ver un dashboard de progreso por estudiante
- [ ] Se muestra porcentaje de aciertos por tema/asignatura
- [ ] Se puede filtrar por período (semana, mes, bimestre)
- [ ] Los datos se generan a partir de los intentos en ejercicios de práctica
- [ ] Funciona offline con datos locales
