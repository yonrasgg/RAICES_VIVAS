## Descripción

Como **docente comunitario**, necesito registrar estudiantes con nombre, nivel, grado, lengua materna y territorio, para llevar control de los alumnos a mi cargo.

## Criterios de Aceptación

- [ ] El docente puede crear un registro de estudiante con campos obligatorios: nombre, nivel, grado, lengua materna, territorio
- [ ] El sistema asigna un ID único al estudiante
- [ ] El registro se almacena localmente y se marca para sincronización
- [ ] El docente puede editar datos del estudiante posteriormente
- [ ] La lista de estudiantes es filtrable por grado y lengua
