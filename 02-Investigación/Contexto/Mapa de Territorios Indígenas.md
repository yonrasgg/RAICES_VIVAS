# 🗺️ Mapa de Territorios Indígenas de Costa Rica

> Este mapa interactivo muestra la ubicación de los **24 territorios indígenas** reconocidos por la legislación costarricense (Ley Indígena N° 6172, 1977). El proyecto **Raíces Vivas** trabaja con estas comunidades para fortalecer educación, saberes ancestrales y salud comunitaria.

## Pueblos Indígenas de Costa Rica

Costa Rica reconoce **8 pueblos indígenas** distribuidos en 24 territorios:

### Bribri (Talamanca)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Talamanca, Limón |
| **Población aprox.** | ~13,000 |
| **Lengua** | Bribri (familia chibchense) |
| **Territorio** | Alto Talamanca, Kéköldi |
| **Relevancia Raíces Vivas** | Módulo Educación: preservación lingüística; Módulo Saberes: medicina tradicional y cosmovisión |

### Cabécar (Chirripó)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Chirripó, Tayní, Telire, Nairi-Awarí, Bajo Chirripó, China Kichá |
| **Población aprox.** | ~17,000 |
| **Lengua** | Cabécar (familia chibchense) |
| **Territorio** | Cordillera de Talamanca (zona más remota) |
| **Relevancia Raíces Vivas** | Módulo Salud: barreras geográficas extremas; Módulo Educación: falta de escuelas |

### Maleku (Guatuso)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Guatuso, Alajuela |
| **Población aprox.** | ~1,000 |
| **Lengua** | Maleku (familia chibchense, en peligro) |
| **Territorio** | Llanuras del Norte |
| **Relevancia Raíces Vivas** | Módulo Saberes: lengua en riesgo de extinción; Módulo Educación: revitalización |

### Boruca (Buenos Aires)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Buenos Aires, Puntarenas |
| **Población aprox.** | ~2,600 |
| **Lengua** | Boruca (prácticamente extinta) |
| **Territorio** | Valle central del Pacífico Sur |
| **Relevancia Raíces Vivas** | Módulo Saberes: Danza de los Diablitos, artesanía en balsa |

### Térraba (Buenos Aires)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Buenos Aires, Puntarenas |
| **Población aprox.** | ~750 |
| **Lengua** | Térraba (extinta como lengua nativa) |
| **Relevancia Raíces Vivas** | Módulo Saberes: recuperación cultural; Módulo Salud: conflictos territoriales |

### Ngäbe-Buglé (Coto Brus)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Coto Brus, Puntarenas (frontera con Panamá) |
| **Población aprox.** | ~10,000 (incluyendo migración estacional) |
| **Lengua** | Ngäbere |
| **Relevancia Raíces Vivas** | Módulo Salud: población migrante, acceso precario a servicios |

### Huetar (Valle Central)

| Aspecto | Detalle |
|---------|---------|
| **Ubicación** | Quitirrisí (Mora) y Zapatón (Puriscal) |
| **Población aprox.** | ~2,500 |
| **Lengua** | Huetar (extinta) |
| **Relevancia Raíces Vivas** | Módulo Educación: integración urbana; proximidad al Valle Central |

### Rey Curré / Guaymí / Otros

Territorios adicionales distribuidos en el Pacífico Sur (Buenos Aires, Osa, Conte Burica, Abrojos Montezuma). Algunos comparten características con pueblos Boruca, Térraba y Ngäbe-Buglé.

## Contexto Legal

|-------------|-------------|
