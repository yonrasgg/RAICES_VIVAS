## Términos del Dominio

| Término | Definición |
|---------|-----------|
| **Administrador comunitario** | Persona designada por la comunidad para gestionar permisos y gobernanza dentro del sistema |
| **Brigada de salud** | Equipo de profesionales que realiza visitas periódicas a territorios indígenas |
| **Consentimiento informado** | Aprobación documentada del portador de saber o comunidad para registrar/compartir contenido |
| **Docente comunitario** | Educador que trabaja en territorios indígenas, frecuentemente bilingüe |
| **Gobernanza cultural** | Modelo de control donde la comunidad define quién accede a qué contenido cultural |
| **Nivel de acceso** | Clasificación de contenido: público, comunitario, restringido o ceremonial/sensible |
| **Modo offline** | Capacidad del sistema de funcionar sin conexión a internet, con sincronización posterior |
| **Portador de saber** | Persona reconocida por la comunidad como fuente de conocimiento ancestral |
| **Saber ancestral** | Conocimiento tradicional transmitido intergeneracionalmente (agricultura, medicina, rituales, etc.) |
| **Sincronización** | Proceso de actualizar datos entre dispositivo local y servidor cuando hay conectividad |
| **Territorio indígena** | Área geográfica legalmente reconocida como territorio de un pueblo indígena en Costa Rica |
| **Trazabilidad** | Capacidad de vincular un requerimiento a su problema de origen, necesidad y criterio de aceptación |
