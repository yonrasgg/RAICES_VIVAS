# Alcance del Proyecto — Raíces Vivas

## Scope Statement

### Dentro del Alcance

| Área                        | Incluido                                                                      |
| --------------------------- | ----------------------------------------------------------------------------- |
| **Análisis**                | Contexto sociocultural, actores, necesidades                                  |
| **Requerimientos**          | 16 RF + 7 RNF definidos y priorizados (MoSCoW)                                |
| **Arquitectura**            | [[WBS]], diagramas de contexto, [[Modelo de Datosmodelo de datos]] conceptual |
| **Validación**              | Entrevistas, encuestas, observación con usuarios potenciales                  |
| **Diseño (Avance 2)**       | Prototipos UI, modelo ER detallado, stack tecnológico                         |
| **Implementación (futuro)** | Módulo piloto MVP                                                             |

### Fuera del Alcance

| Área | Excluido | Razón |
|------|----------|-------|
| Implementación completa | Los 3 módulos en producción | Complejidad del curso |
| Certificación MEP | Validación curricular oficial | Requiere proceso institucional |
| Expediente clínico hospitalario | Integración con CCSS | Fuera de jurisdicción |
| IA / ML | Indexación avanzada de saberes | Complejidad técnica excesiva para MVP |
| Identidad estatal | Integración con sistemas gobierno | Fuera del alcance académico |

## Criterios de Éxito

1. ✅ Requerimientos definidos con criterios medibles y verificables
2. ✅ Trazabilidad completa: Problema → Necesidad → [[WBS]] → RF/RNF → Criterio de aceptación
3. ⬜ Validación con al menos 2-3 usuarios reales
4. ⬜ Diseño de arquitectura modular y escalable
5. ⬜ Prototipo funcional de al menos 1 módulo

## Supuestos

- Se logrará acceso a usuarios potenciales (docentes, personal salud) para validación
- La conectividad permitirá sincronización eventual (no requiere tiempo real)
- Los líderes comunitarios participarán en definición de gobernanza

## Restricciones

- Plazo del curso académico (semestre)
- Equipo de 3 integrantes
- Sin presupuesto externo
- Sensibilidad cultural (consentimiento informado obligatorio)
